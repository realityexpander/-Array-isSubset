# -Array-isSubset
<br>
Create a function that takes two arrays and determine whether the second array is a subset of the first array. Arrays will only contain primitive values.<br>
<br>
Examples<br>
Input	Output<br>
array:<br>
[ 1, 2, 3, 4 ]<br>
sub:<br>
[ 2, 3, 4 ]	true<br>
array:<br>
[ 2, 2, 2, 3, 4 ]<br>
sub:<br>
[ 2, 4, 3 ]	false<br>
array:<br>
[ 2, 3, 3, "a" ]<br>
sub:<br>
[ 3, 3, 3, "a" ]	false<br>
